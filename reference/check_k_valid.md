# Check if k is valid

Check if k is valid

## Usage

``` r
check_k_valid(k, max_k, arg = rlang::caller_arg(k), call = rlang::caller_env())
```

## Arguments

- k:

  Number of components

- max_k:

  Maximum allowed components

- arg:

  Argument name for error messages

- call:

  Calling environment for error messages
